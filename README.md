# 📂 Tic-Tac-Toe Game  

This repository contains my **Tic-Tac-Toe game application** built using Java and XML in Android Studio. It is a classic two-player strategy game where users compete to align their symbols in a row.

---

## 📚 About

The purpose of this project was to practice Android development and game logic implementation by building a fully functional Tic-Tac-Toe game from scratch.

---
 
## 🚀 Features

* Two-player gameplay
* Turn-based play system
* Win / Draw detection
* Game reset option 

---

## 🛠️ Tech Used

* Java
* XML
* Android Studio

---

## 🎯 Learning Outcome

Through this project, I learned:

* Game logic implementation
* Handling multiple user turns
* UI interaction handling
* Grid layout design in Android

---

## 🌟 Project Effort

This project required significant effort in designing the game flow and implementing win-condition logic, making it an important part of my learning journey.
